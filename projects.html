Face Recognition System

Live

I developed a Face Recognition System using Python, OpenCV, and machine learning concepts. The project captures real-time video from a webcam, detects faces using Haar Cascade Classifier, and recognizes individuals using LBPH (Local Binary Patterns Histogram) algorithm. The dataset is generated by capturing multiple images of a person, training the model, and then performing recognition with high accuracy. This project focuses on automation and security, where the system can identify authorized users and deny unknown faces. It includes features like face detection, data preprocessing, model training, and real-time recognition. This project improved my understanding of computer vision, image processing, Python libraries, and real-time data handling.

swiftcart

Live

SwiftCart is a modern, full-featured e-commerce platform developed using PHP, MySQL, JavaScript, and Tailwind CSS. The project aims to deliver a seamless online shopping experience for users, featuring a robust product catalog, dynamic cart and wishlist management, secure checkout, and comprehensive order tracking. SwiftCart supports multiple user roles, including customers, vendors, and administrators, each with tailored dashboards and functionalities. The platform emphasizes responsive design, intuitive navigation, and real-time interactivity through AJAX-powered components. Security, scalability, and maintainability are core principles guiding the development, making SwiftCart suitable for deployment in real-world retail environments
